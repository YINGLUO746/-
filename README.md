超市管理系统 (Supermarket Management System)
基于 C开发的简易超市管理系统，实现商品信息管理、记录库存、结算记账等核心功能。

项目简介
本项目是一款面向小型超市的管理系统，采用 C 语言开发，核心功能涵盖商品基础信息维护、库存动态管理、结算记账等模块，旨在简化超市日常运营流程，提升管理效率。

核心功能
1.商品管理：商品信息的添加（目前只有这个功能）
2.库存管理：实时库存数量监控，动态变化
3.购物功能：商品买入结算功能，购物车功能实现
4.数据持久化：通过文件存储商品及销售数据，确保数据不丢失

环境依赖
开发工具：Visual Studio 2022 及以上版本
编译环境：支持 C99 标准的 C 语言编译器
系统环境：Windows 7/10/11
